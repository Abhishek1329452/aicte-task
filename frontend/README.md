Frontend (Vite + React) quick start

1. Install dependencies:

```bash
cd frontend
npm install
```

2. Start dev server:

```bash
npm run dev
```

3. By default the frontend will POST to `http://localhost:8000`. To change, set `VITE_API_BASE` in a `.env` file in the `frontend` folder, e.g. `VITE_API_BASE=https://your-backend.example`.
